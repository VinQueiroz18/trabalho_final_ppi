Sistema de Gerenciamento de Biblioteca

Aplicação web desenvolvida com Node.js e Express para controle de livros e leitores.

🚀 Funcionalidades
- Cadastro de livros
- Cadastro de leitores
- Sistema de login com sessão
- Controle de empréstimos
- Exibição do último acesso com cookies

🛠️ Tecnologias
- Node.js
- Express
- Cookies e Sessões
